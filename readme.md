# 📄 Project Closure Document Generator

A Streamlit-based tool to generate structured Project Closure Documents.

## 🚀 Features
- Guided UI with tooltips and help sections
- Full document preview
- File uploads (images embedded)
- Custom sections
- Stakeholder management
- Word document generation with TOC

## ▶️ Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
